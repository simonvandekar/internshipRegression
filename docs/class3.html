Introduction

Let’s use the insurance dataset to investigate the following questions.

  1. Age Influence on Health Charges:
    • How does age affect medical charges? is the relationship linear or nonlinear?
  2. Influence of Smoking on Medical Expenses:
    • How does smoking status influence medical charges, and are smokers subjected to significantly higher health-related expenses compared to non-smokers?
  3. Does smoking affect the charges age trajectory: Is there evidence that smoking changes the effect of age on charges?
  4. Regional Variations in Healthcare Charges:
    • Are there regional differences in medical charges, and how do regional factors potentially contribute to the variation in healthcare costs?
  5. BMI:
    • Does BMI relate to insurance charges? Is it nonlinear?
  6. Combined Effects of Predictors:
    • What does the relationship between each variable look like after controlling for the others? Hint: Use added variable plots.
    • Does smoking status affect the influence of age on medical charges?

Methods

Describe the overall strategy:

  • For all study aims, use charges as the outcome including age, sex, region, bmi, and smoking status as covariates (independent variables).

Age Influence on Health Charges:

To evaluate the effect of age on medical charges, we performed a test of the linear age effect using a T-test and report the effect with the estimate, confidence interval, T-statistic, degrees of freedom, and \(p\)-value.

There was evidence that age was positive associated with medical charges (b=258.54, StdEr=11.94, \(T=21.66\), \(df=1300\), \(p <0.001\)).

# Get predictions for both smoker groups
ageGrid <- ref_grid(logpolyInt, at = list(age = seq(min(ins$age), max(ins$age), length.out = 100)))
predictions <- as.data.frame(emmeans(ageGrid, ~ age | smoker))

# Back-transform to original scale for BOTH groups
predictions[, c('emmean', 'lower.CL', 'upper.CL')] <- 
  exp(predictions[, c('emmean', 'lower.CL', 'upper.CL')])

# Make smoker a factor for clean labels
predictions$smoker <- factor(predictions$smoker, levels = c(0, 1), labels = c("Non-smoker", "Smoker"))
ins$smoker_f       <- factor(ins$smoker,         levels = c(0, 1), labels = c("Non-smoker", "Smoker"))

# Plot both groups
ins$region = as.factor(ins$region)
ggplot() +
  geom_point(data = ins, aes(x = age, y = charges, color = region), alpha = 0.4) +
  geom_ribbon(data = predictions, aes(x = age, ymin = lower.CL, ymax = upper.CL, fill = smoker), alpha = 0.25) +
  geom_line(data = predictions, aes(x = age, y = emmean, color = smoker), linewidth = 1.2) +
  scale_color_manual(values = c("northeast" = "blue", "northwest" = "red", 'southeast' = "yellow", 'southwest'='green')) +
  scale_fill_manual(values = c("northeast" = "blue", "northwest" = "red", 'southeast' = "yellow", 'southwest'='green')) +
  labs(title = "Charges vs. Age: Smoking × Age Interaction",
       subtitle = "Cubic polynomial interaction (log-scale model, back-transformed)",
       x = "Age", y = "Charges",
       color = "Smoking Status", fill = "Smoking Status") +
  theme_minimal()
